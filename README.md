# 우리 FISA Frontend Toy Project

## 💵 서비스 소개
우리 FISA 서비스 개발 과정 토이프로젝트로 html, css와 js를 활용한 가계부 페이지입니다.

### 📍 site
https://woorifisa-service-dev-4th.github.io/frontend-1st-woori-account-book/

</br>

### 📍 Branch Name Rule
아래와 같이 통일
```
- main
- dev
- feat/#{issue_number}
```

</br>

### 📍 Commit Message Convention

|    Type     | Description  |
|:-----------:|---|
|   `Feat:`   | 새로운 기능 추가 |
|   `Fix:`    | 버그 수정 |
| `Refactor:` | 코드 리팩토링 |
| `Rename:` | 파일 혹은 폴더명을 수정할 때 사용|
| `Remove:` | 파일을 삭제할 때 사용 |

➡️ Feat: button 추가

</br>

### 📍 Class Name Rule
- 레이아웃(align / position) :  `layout_{ }`
- 컨테이너(wrapper) : `container_{ }`
- 글꼴 : `text_{ }`
- 이미지 : `img_{ }`
- 글꼴색 : `font_color_{ }`
- 배경색 : `bg_color_{ }`
- 구분선 : `line`

</br>
